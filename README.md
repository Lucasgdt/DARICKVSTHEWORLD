# DARICKVSTHEWORLD < HIGHWAY TO L2

Bonjour ! Bievenue sur notre jeu nommé : DARICK vs THE WORLD QUI EST MOINS BIEN QUE HIGHWAY TO L2

# Introduction 

DARICK vs THE WORLD est un jeu type "rogue like" dont le but est d'aller de salle en salle en tuant chaques monstres qui s'y trouve en moins bien que HIGHWAY TO L2
L'objectif du joueur, aller le plus loins possible dans les salles en ayant tuer le plus de monstres possible ! (... toujours pour la performance)
Ce jeu a été developpé en langage C ainsi qu'en SDL2 pour l'affichage et le visuel (un peu comme tout le monde mais bref...).
Le projet a été developpé par GAUDREE Mathéo (qui a fais que l'invetaire), GOUDET Lucas & LAHMAR Marwan

# Fonctionnement du jeu

Afin de pouvoir utiliser notre jeu, veuillez au préalable installer SDL2 ainsi que les extentions suivantes : Image & ttf. (AH BON on avais y avais pas pensé...)
Une fois cela fait, vous pouvez compiler notre jeu à l'aide du Makefile !(il marche pas...) (N'oubliez pas également de modifier le chemin vers la SDL2 dans ce même Makefile)

# Utilisation

Dès que la compilation sera terminé, le jeu se lancera automatiquement !
(mais QUI a envie d'y joué ?)

#  Quels sont les fonctionnalités du jeu ?

Le jeu possède une génération et une apparition des mobs aléàtoirement. (heureusement c'est la base...) 
Le joueur peut attaquer les mobs qui s'y trouvent sur son passage afin d'obtenir des nouveaux objets qui sont plus ou moins rare, ainsi qu'une chance d'obtenir de l'argent.
Vous possèdez un inventaire utilisable à n'importe quel moment de la partie 
Si vous mourrez, vous perdez tout, donc faites attention !
